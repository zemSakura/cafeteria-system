| 类别        | 需要确认的配置                  | 推荐值 / 示例                                          | 你要去找的信息                           |
| --------- | ------------------------ | ------------------------------------------------- | --------------------------------- |
| 操作系统      | 部署系统版本                   | Windows 10 / Windows 11                           | 虚拟机里安装的系统版本                       |
| 系统位数      | 32 位还是 64 位              | 64 位                                              | 设置 → 系统 → 关于                      |
| CPU       | 虚拟机 CPU 核心数              | 2 核及以上                                            | 虚拟机设置里查看                          |
| 内存        | 虚拟机内存大小                  | 4GB 以上，推荐 8GB                                     | 虚拟机设置里查看                          |
| 硬盘空间      | 虚拟机磁盘大小                  | 30GB 以上                                           | 虚拟机设置里查看                          |
| Java 环境   | JDK 版本                   | JDK 17 或 JDK 21                                   | 命令行输入 `java -version`             |
| Java 安装路径 | JDK 安装目录                 | 如 `C:\Program Files\Java\jdk-17`                  | 环境变量或安装目录                         |
| JAVA_HOME | Java 环境变量                | 指向 JDK 安装目录                                       | 系统环境变量里查看                         |
| Path 配置   | 是否加入 Java bin 目录         | `%JAVA_HOME%\bin`                                 | 系统环境变量 Path                       |
| 开发工具      | IDE 版本                   | IntelliJ IDEA 2023/2024/2025 均可                   | IDEA 关于页面查看                       |
| 项目 SDK    | IDEA 使用的 JDK             | JDK 17 或 JDK 21                                   | IDEA Project Structure 查看         |
| 编译方式      | 普通 Java / Maven / Gradle | 看项目结构决定                                           | 项目里有没有 `pom.xml` 或 `build.gradle` |
| 项目路径      | 项目所在目录                   | 如 `D:\Canteen-Simulation`                         | 你把项目放在哪里                          |
| 主程序入口     | 启动类                      | 如 `Main.java` / `App.java` / `SimulatorMain.java` | 找含有 `public static void main` 的类  |
| 数据库       | 是否使用数据库                  | 你们项目大概率“不使用数据库”                                   | 看代码里有没有 MySQL、JDBC、数据库配置          |
| 数据库版本     | 如果用了数据库                  | MySQL 8.0                                         | 没用就写“无”                           |
| 第三方依赖     | 是否需要额外依赖                 | Java 标准库 / Maven 依赖                               | 看项目是否需要下载依赖                       |
| 前端环境      | 是否有前端页面                  | JavaFX / Swing / Web 前端 / 控制台                     | 看你们系统启动后的界面形式                     |
| 浏览器       | 如果是 Web 页面               | Chrome / Edge                                     | 如果不是网页可写“不需要”                     |
| 端口号       | 如果是 Web 系统               | 如 `8080`                                          | 看代码配置或启动日志                        |
| 配置文件      | 系统参数配置位置                 | 如 `CanteenConfig.java`                            | 找配置类或配置文件                         |
| 仿真参数      | 人数、窗口数、桌子数、时间等           | 根据系统默认值填写                                         | 看 `CanteenConfig` 或参数设置界面         |
| 运行命令      | 启动系统命令                   | `java -jar xxx.jar` 或 IDEA 运行                     | 根据实际运行方式填写                        |
| 打包文件      | 是否生成 jar                 | 如 `canteen-simulation.jar`                        | 如果你们需要打包，就记录 jar 名称               |
| Git 工具    | 是否需要 Git                 | 可选                                                | 如果从仓库拉代码，需要安装 Git                 |
| 网络环境      | 是否需要联网                   | 安装依赖时需要                                           | 部署后一般不需要                          |
